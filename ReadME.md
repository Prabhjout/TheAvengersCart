
# Avengers11Cart

Its a GUI for Shopping made using JavaFx and basic Principles of OOPS.


## Tech

JavaFx


## Features

- Minimizing the number of bags Required for items Purchased.
- Items Content in each bag and wait of each bag.
- Billing.
